Comparison Discription Syntax
== Equality Operator compare two values and return a boolean.
Type coercion: If compare two different data types,
its will convert one type to another in order to compare.		 2 == 3 // false
1 == '1' // true
=== Strict equality Operator compare two values and return a boolean.
There is no type coercion.		 2 === "2" // false
1 === 1 // true
!= Inequality operator != is the opposite of the equality operator.
It will convert data types of values while comparing (type coersion)		 "1" != 1 // false
2 != 5 // true
0 != false // false
!== Strict inequality operator !== is the opposite of the strict equality operator.
There is no type coercion.		 2 !== "2" // true
1 !== 1 // false
