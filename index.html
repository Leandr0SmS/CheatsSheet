CSS

Selector Discription Example
# Select a element by id to add style. #elementId { property: value; }
. Select a element by class to add style. .className { property: value }
Element tag without <> Select a element to add style. p { property: value }
Element Discription Example
background The background property sets all background style properties at once,
such as color, image, origin and size, or repeat method.		 background: green; => Change the background color.
background: url("path to file") => Change the background to a image.
border The border property sets an element's border.
It sets the values of border-width, border-style, and border-color.		 border: 2px solid green; => Add a green 2px solid border to the element.
